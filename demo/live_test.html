What we are spoofing & drifting:
  1. Canvas 2D Hash: We inject deterministic pixel noise invisible to the human eye. The hash of the pixels changes on every reload.
  2. WebGL Image Hash: We intercept readPixels to inject noise in the 3D render buffer. The hash changes on every reload.
  3. Audio Buffer Hash: We intercept getChannelData in the Web Audio API and add micro-fluctuations. The hash changes on every reload.
  4. WebGL Metadata (Params): We mask the graphics card vendor/renderer strings so trackers see spoofed hardware. This stays stable per session.
